Cavaleiros Marginais

Este projeto foi desenvolvido com o objetivo de divulgar a banda Cavaleiros Marginais.  
Todo o design e programação foram feitos por mim.

Jogo Cavaleiros Marginais
- Para acessar, basta executar o arquivo executável localizado na pasta site/cavaleiros marginais.  
- O jogo foi criado originalmente em Adobe Flash.  
- Por conta do tamanho dos arquivos, o jogo está sem som, já que os áudios precisariam estar anexados junto ao arquivo principal e optei por deixá-lo mais leve.

Conteúdo adicional
- Dentro da pasta site está também o site antigo que desenvolvi junto com o projeto.  
- Na pasta rasante de fogo encontra-se o protótipo de outro jogo.  
  - Devido à descontinuação do Flash, este jogo não funciona mais diretamente.  
  - Apenas o executável protótipo pode ser rodado, apresentando alguns bugs.

Observações
- O projeto reflete uma época em que o Flash era amplamente utilizado para jogos e sites interativos.  
- Hoje, os arquivos .swf não são mais suportados em navegadores modernos, mas os executáveis ainda podem ser abertos em sistemas Windows.
